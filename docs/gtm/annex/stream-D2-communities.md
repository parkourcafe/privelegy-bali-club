Start with these three:
1. **TripAdvisor Bali Forum** — create a personal (non-brand) account and answer 3–5 open "which area/where to eat" questions per week with genuinely specific local answers, zero links, for 3–4 weeks before any mention of Other Bali; promo is effectively banned, so this channel is for credibility and demand-language research only.
2. **r/bali** — same value-first pattern (answer itinerary/area-fit questions in comments, no links), then modmail the mods asking permission for a single "free structured Bali decision tool" post; do not post any link before explicit mod approval.
3. **Bali Bogans** — join with a personal profile, contribute helpful AU-relevant answers for 2+ weeks, then message the admins proposing an admin-approved pinned resource ("free English guide: which Bali area fits your trip"), explicitly offering it with no booking/payment angle.
