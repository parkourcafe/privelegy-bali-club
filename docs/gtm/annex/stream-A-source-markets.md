## 4. Booking / planning window — [WEAK EVIDENCE]

- Australia: consumer-finance/travel guidance (not official statistics): optimal flight-booking window **6–14 weeks** pre-departure; **12–20 weeks** for peak periods (Jul + Dec–Jan school holidays); one guide advises 5–7 months for an October Bali trip; accommodation 3–4 months ahead for Jul–Aug.
