# Getting Other Bali onto Google's first page — 20 actions

Prioritised by impact. Honest framing: on-page is largely done; the real
needle-movers for a new domain are **backlinks + more category content +
internal linking + photos + time**. No list makes a brand-new site rank in
days. Win the long-tail first (specific venue + intent), then climb.

Owner key: **[me]** = code/content I ship · **[you]** = accounts/outreach only
you can do · **[both]**.

---

## Tier 1 — Biggest impact (mostly you)

1. **Backlink wave to venues** [you] — get featured venues to link their
   `/places/<slug>` page from their own site. Each link is authority we can't get
   otherwise. Templates + the 12-venue pack are ready (`docs/venue-outreach.md`).
2. **GSC Domain property + resubmit sitemap + Request indexing** [you] — proper
   property (covers www + apex), resubmit `sitemap.xml`, request indexing on the
   new URLs. Speeds Google's pickup from weeks to days.
3. **Stabilise the domain** [you] — set `otherbali.com` as the primary domain on
   Vercel (one canonical host, www → apex) so crawl never breaks again.
4. **Real photos on venue pages** [you/creators] — trust, engagement, and image
   search. Currently venue pages have no photos; adding them is a big quality lift.
5. **Digital PR / citations** [you] — get Other Bali mentioned/linked on
   established Bali sites, expat groups, and "best Bali guides" round-ups.
6. **Brand entity presence** [you] — consistent Other Bali social profiles + a
   consistent name/description everywhere, so Google builds an entity for the brand.

## Tier 2 — Content & internal linking (me)

7. **Auto-link venue names in guides** [me] — where a guide names a real venue,
   link it to its `/places` page. More internal links → those pages rank better.
8. **Link guides from the pillars** [me] — e.g. `/canggu` → `canggu-vs-uluwatu`,
   `best-coffee-in-bali`; `/uluwatu` → `best-beach-clubs-in-bali`.
9. **Site-wide link to `/guides`** [me] — in the header/footer so every page
   passes crawl equity into the guides cluster.
10. **Homepage → top guides + pillars** [me] — surface the strongest pages from
    the highest-authority page on the site.
11. **New category spokes for high-volume queries** [me] — e.g. "best warungs in
    Canggu", "best brunch in Canggu", "cheap eats in Canggu". These have real
    search volume and we have the venue data.
12. **Editorial for near-ready venues** [me] — finish the few remaining thin
    active rows so more pages become indexable.
13. **Freshness** [me] — keep publishing/updating; Google favours live, growing
    sites.

## Tier 3 — Technical & AEO (me)

14. **Title/meta tuning for target keywords** [me] — front-load the keyword,
    write click-worthy descriptions (lifts CTR, which lifts ranking).
15. **Richer venue structured data** [me] — `sameAs` (official site + IG) +
    `priceRange` now ship for every district (done); next: verified hours and
    menu links where we have them.
16. **FAQPage schema on pillars & category pages** [me] — mirror real questions;
    wins AI answers and rich results.
17. **Core Web Vitals / speed** [me] — keep LCP fast, images optimised; page
    experience is a real ranking factor. Verify with Lighthouse on prod.
18. **Index hygiene** [me] — thin/unverified pages stay `noindex` (publication
    gate already enforces this); keep the sitemap to only indexable URLs so crawl
    budget goes to pages that can rank.

## Tier 4 — Measure & iterate (you give access, I analyse)

19. **Query mining from GSC** [both] — read the real queries we get impressions
    for, and build a dedicated spoke for each recurring "best X in {area}" with no
    page yet. Highest-ROI new pages, because demand is proven.
20. **Work the "striking distance" pages** [both] — filter GSC positions 5–15
    (already ranking, one nudge from page one) and improve those specific pages
    first: better title, more internal links, a sharper answer-first opening.

---

## The honest timeline
- Weeks: long-tail + branded venue queries start converting impressions to clicks.
- 1–3 months: category queries ("best X in {area}") climb as internal links +
  the first backlinks land.
- 3–6+ months: head terms move, gated by domain authority (backlinks) and time.

The single biggest lever is **#1 (venue backlinks)** — everything on-page is
necessary but not sufficient without external signals.
