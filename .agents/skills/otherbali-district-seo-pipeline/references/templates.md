# District document templates

## Unified decision row

Include: topic; status; exact canonical; create/update/merge target; primary intent; user decision; overlap; cannibalization risk; evidence readiness; owner dependency; field dependency; Maps dependency; GSC dependency; schema; links in/out; publication gate; priority; confidence; extracted/interpreted basis.

## P0 brief

Include: URL; primary intent; excluded intents; user outcome; H1; metadata; section outline; allowed claim IDs by section; prohibited wording; schema; internal links; verified date; publication gates.

## Final report

Include: pipeline artifacts; agents/review roles; existing pages; decision matrix; implemented P0 pages; claim IDs; blockers; verification packs; validation results; branch; commit; preview; diff summary; next queue item.

## Hold record

Include: topic; blocker type; missing evidence or confirmation; exact closure action; responsible human/system; earliest next decision; prohibited work while held.
